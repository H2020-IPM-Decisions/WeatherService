IPM Decisions Weather service

This service is part of the Horizon 2020 IPM Decisions project. The intention is to provide the system with sufficient information for a client to be able to connect to and get information from any weather data source

The source code for this service can be found here: https://github.com/H2020-IPM-Decisions/WeatherService

Services

  • List of weather parameters
  • Json schema defining the platform's format for weather data exchange
  • List of available weather datasources
  • Sample weather data output: Forecasts from Met Norway
